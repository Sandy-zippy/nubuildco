# NOTES FOR MEDIA BUYER

1. **Primary Text Truncation:** Meta truncates at ~125 characters on mobile feed. The first line of every variant is crafted to hook within that limit. Always preview on mobile.
2. **Headline Length:** All headlines are within 40 characters. Shorter headlines (under 30) tend to perform better on mobile placements.
3. **Description:** Descriptions only appear on certain placements (desktop feed, right column). Keep them but do not rely on them for message delivery.
4. **A/B Testing Priority:** Test hook styles against each other first (Question vs. Fear vs. Authority), then test variants within the winning hook style.
5. **Offer Ladder:** Cold audiences should see Cost Guide or Strategy Call offers first. Warm/retargeting audiences should see the ₹4,999 Design Consultation.
6. **Dynamic Creative:** All variants are designed to be modular. You can mix primary texts with headlines from different variants within the same hook style.
7. **Emoji Policy:** No emojis are used in this copy bank. The brand tone is premium and text-only. If testing emojis, limit to one per primary text — never in headlines.
8. **Landing Page URL:** https://nubuildco.netlify.app — append UTM parameters per variant (e.g., `?utm_source=meta&utm_medium=paid&utm_campaign=design_consultation&utm_content=fear_budget_2a`).
